**🎤 全网唯一免费的语音跟踪提词器 —— 说到哪，滚到哪**

免费开源的网页提词器，自媒体博主拍视频必备。

**在线使用 👉 https://jzocb.github.io/teleprompter/**

## ✨ 核心功能

### 🎤 语音跟踪（独家功能）
- **说到哪，滚到哪** - 自动识别你说的话，滚动到对应位置
- 支持中文、英文等多语言
- 说快了？它跟上。暂停了？它等你。

### 📹 相机模式
- 边看提词器边预览自己
- 文字叠加在相机画面上
- 录制时只看一个屏幕

### 🎨 更多功能
- 📏 **字体大小可调** - 24px 到 72px，适配不同距离
- 🚀 **滚动速度可调** - 播放时也能实时加减速
- ⏱️ **倒计时开始** - 3秒/5秒倒计时，给你准备时间
- 🪞 **镜像模式** - 配合镜子或提词器玻璃使用
- 🎨 **7种主题** - 适配不同环境光线
- ⏸️ **点击暂停** - 点屏幕任意位置暂停/继续

## 📱 使用方法

1. 打开 https://jzocb.github.io/teleprompter/
2. 粘贴你的视频文案
3. 调整字体大小和滚动速度
4. 点击「开始提词」
5. 把手机/iPad架在镜头旁边开拍！

**小技巧：** 在 Safari 点「分享 → 添加到主屏幕」，下次像 App 一样打开。

## 🛠️ 本地运行

```bash
git clone https://github.com/jzOcb/teleprompter.git
cd teleprompter
# 直接用浏览器打开 index.html 即可
open index.html
```
